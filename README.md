# 📊 EdTech Course Analysis Dashboard

## Overview

The **EdTech Course Analysis Dashboard** is a Business Intelligence project developed using **Power BI** to analyze online learning platforms and recorded courses. The dashboard provides valuable insights into learner engagement, course popularity, instructor performance, language preferences, subtitle availability, and skill demand.

The objective of this project is to transform raw educational data into meaningful business insights that help EdTech companies make informed decisions about content strategy, course development, and learner engagement.

---

# Project Summary

This project analyzes online courses across multiple categories and sub-categories using interactive Power BI dashboards. It focuses on identifying viewer trends, language distribution, course popularity, instructor ratings, and the impact of subtitles and course duration on learner engagement.

The dashboard enables users to interact with the data through dynamic filters and visuals, making it easier to discover patterns and business opportunities.

---

# Problem Statement

The objective of this project is to help an EdTech startup analyze recorded lecture data and answer the following business questions:

- Analyze course distribution across categories and sub-categories.
- Calculate average views for categories, languages, and sub-categories.
- Identify the most demanded skills across different domains.
- Analyze language distribution of courses.
- Find language preferences for the top-performing categories.
- Study the impact of subtitles on viewer engagement.
- Identify top-rated instructors in each category.
- Analyze the relationship between course duration and number of viewers.
- Evaluate how skill diversity affects overall course popularity.

# Key Insights

- Courses are significantly more popular than Specializations and Professional Certificates.
- English is the dominant language for online learning content.
- Machine Learning, Python, Data Analysis, Programming, and Communication are among the most demanded skills.
- Courses with subtitles generally receive higher viewer engagement.
- Highly rated instructors consistently attract more learners.
- Medium-duration courses tend to perform better than extremely long courses.
- Data Science and Computer Science categories generate the highest average viewership.
